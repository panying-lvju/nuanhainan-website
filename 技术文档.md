# 暖海南资讯网站 - 技术文档

## 技术架构

纯静态网站，Nginx托管，零后端依赖。

```
架构：
浏览器 → Nginx(47.106.78.60) → 静态文件(/var/www/nuanhainan/)
                                    ├── index.html
                                    ├── css/style.css
                                    ├── js/app.js + markdown.js
                                    └── data/
                                        ├── index.json (文章索引)
                                        └── articles/
                                            ├── yanglao/*.md
                                            ├── ai-agent/*.md
                                            └── hainan/*.md

内容更新链路：
小扣生成日报 → 转公开链接 → 服务器sync_content.sh定时拉取 → 覆盖data/目录 → 网站自动展示
```

## 核心实现

### 1. 手写Markdown解析器 (js/markdown.js)
- 不依赖第三方库（marked.js等）
- 支持：标题(h1-h6)、段落、粗体、斜体、无序列表、有序列表、引用块、代码块、行内代码、水平线、链接
- 渲染逻辑：逐行扫描 → 识别块级元素 → 处理行内格式 → 输出HTML

### 2. 按需加载 (js/app.js)
- 页面加载时 fetch index.json 获取文章列表
- 点击文章时 fetch 对应 .md 文件
- 浏览器缓存已加载的文章
- 栏目切换不需要刷新页面

### 3. 内容同步 (sync_content.sh)
- 从预配置的公开URL下载最新 .md 文件
- 自动备份旧文件（保留7天）
- 日志记录同步状态
- crontab每天8:00和20:00执行

### 4. Nginx配置
```nginx
server {
    listen 80;
    server_name _;
    root /var/www/nuanhainan;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }

    # 缓存静态资源
    location ~* \.(css|js|png|jpg|gif|ico)$ {
        expires 7d;
    }

    # .md文件直接可访问
    location /data/ {
        default_type text/plain;
    }
}
```

## 数据格式

### index.json
```json
{
  "yanglao": [
    {
      "date": "2026-07-03",
      "title": "文章标题",
      "summary": "一句话摘要",
      "file": "2026-07-03.md"
    }
  ],
  "ai-agent": [...],
  "hainan": [...]
}
```

### 文章 .md 文件格式
```markdown
# 文章标题

> 来源：xxx | 日期：2026-07-03

正文内容...

**重点内容**用粗体标注

- 列表项1
- 列表项2

---

*来源链接*
```

## 同步链接配置

小扣每天生成内容后，会更新以下4个公开下载链接：

| 变量名 | 对应内容 | 状态 |
|--------|---------|------|
| YANGLAO_URL | 养老动态 .md | 待配置 |
| AI_AGENT_URL | AI动态 .md | 待配置 |
| HAINAN_URL | 海南热点 .md | 待配置 |
| INDEX_URL | index.json | 待配置 |

配置方式：编辑 /var/www/nuanhainan/sync_content.sh 中的URL变量

## 运维命令

```bash
# 手动同步
sudo /var/www/nuanhainan/sync_content.sh

# 查看同步日志
tail -30 /var/www/nuanhainan/sync.log

# 查看Nginx状态
sudo systemctl status nginx

# 查看Nginx错误日志
sudo tail -30 /var/log/nginx/error.log

# 修改同步时间
crontab -e

# 重启Nginx
sudo systemctl restart nginx
```
